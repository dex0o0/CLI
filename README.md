**how to install**
for install on linux type in terminal 
> برای نصب این دستورات رو در ترمینال وارد کنید

```bash
cd "address clone git"/kali
chmod +x install.sh
./install.sh #run file install
```
**how to use**
> با زدن این کد میتوانید دستور عمل های کد رو بخونید
```bash
kali --help
kali --version
```
> command wifi
```bash
kali wifi list
kali wifi connect <name_wifi>
kali wifi connection
kali wifi disconnect <name_wifi>
```
